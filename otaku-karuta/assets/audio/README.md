# 読み上げ音声（ハードモード用）

このディレクトリには、ハードモードで上の句を読み上げる音声ファイルを置きます。

- ファイル名: `{id}.wav`（`data/memes.json` の `id` に対応。例: `1.wav`, `2.wav` …）
- 生成方法: VOICEVOX を起動した状態で `../tools/generate_voices.ps1` を実行
  （詳細はスクリプト冒頭のコメント参照）

## 補足

- 音声ファイルが無い問でも、ゲーム側のフォールバックにより上の句テキストが表示され、
  プレイは継続できます（読み上げが無いだけ）。
- ライセンス: VOICEVOX の生成音声を公開する場合は、使用キャラクターのクレジット表記
  （例: `VOICEVOX:ずんだもん`）が必要です。`index.html` 内に記載済み。
